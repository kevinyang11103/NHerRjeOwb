## 技术栈

- 后端框架：Django
- 数据库：MySQL
- 编程语言：Python

## 功能模块

1. 用户管理
   - 用户注册与登录
   - 用户信息管理
2. 课程管理
   - 课程发布与更新
   - 课程分类与检索
3. 学习进度跟踪
   - 课程学习记录
   - 作业提交与批改

   - 实时消息通知
